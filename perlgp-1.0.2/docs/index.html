To help you use PerlGP, the following resources are available:

  • PerlGP manual online - HTML format, recommended (opens in separate window because there are no backward links)
  • PerlGP manual PDF download - more than 60 pages, please think of the trees before you print all of it!
  • PerlGP manual Postscript format download - see above. Here's a double sided two-up version which will save some paper if you send it to a duplex postscript printer
  • The LaTeX source file for the manual are included with the standard distribution
